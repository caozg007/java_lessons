1.学会问机器
	机器是最真实的，最正确的。如果你遇到一些语句不确定结果，记得把它运行看看结果。
	比如说：int a = 1; int b = ++a + a++ * (a--+1); b的值是多少？只需要把这一行代码放到程序里，运行一遍看结果。
2.看完题目，理清思路，设计解决方案，最后才动手。
	不要在一知半解的时候就动手写代码，鲁莽的动手结果往往是需要重写代码。
3.学会抽取方法
	同样的代码，复制粘贴3遍到不同的地方？这时你需要写一个方法，在3个不同的地方调用这个方法。
4.学会在网上找答案
	遇到什么不懂的，可以去baidu、google寻找答案。学会快速的从网页中获取你想要的知识。
